# ISO Tracker Brand Style Guide
**Interstellar Object Analysis Platform**

*Version 2.0 | Last Updated: November 9, 2025*

---

## Brand Essence

**"Revealing Interstellar Truth"**

Our brand exists at the intersection of scientific rigor and cosmic wonder. We provide evidence-based analysis platforms that help anyone evaluate interstellar object claims with the same rigor as scientific consensus—making the pursuit of interstellar knowledge accessible, trustworthy, and inspiring.

### Brand Promise
**"Track interstellar visitors. Decide if we're alone."**

### Core Value Proposition
**"See the evidence. Weigh the expert takes. Cast your view—alien or natural?"**

We are the only evidence-based analysis platform dedicated exclusively to interstellar objects, bridging the gap between speculative social media and inaccessible academic sources.

## Color System

### Primary Palette

#### Cosmic Deep Blue (Primary Brand Color)
- **Hex:** `#0A1628`
- **RGB:** 10, 22, 40
- **Usage:** Primary backgrounds, navigation headers
- **Meaning:** Depths of space, scientific authority
- **Accessibility:** White text achieves WCAG AAA (16.8:1 contrast)

#### Starlight White (Primary Text)
- **Hex:** `#F5F7FA`
- **RGB:** 245, 247, 250
- **Usage:** Primary text, light backgrounds
- **Meaning:** Clarity, starlight, precision
- **Accessibility:** Black text achieves WCAG AAA (18.2:1 contrast)

#### Nebula Blue (Interactive Elements)
- **Hex:** `#2E5BFF`
- **RGB:** 46, 91, 255
- **Usage:** Links, buttons, CTAs, active states
- **Meaning:** Discovery, interaction, cosmic phenomena
- **Accessibility:** WCAG AAA compliant on dark backgrounds

#### Trajectory Gold (Accent & Highlights)
- **Hex:** `#FFB84D`
- **RGB:** 255, 184, 77
- **Usage:** Orbital paths, highlights, featured content
- **Meaning:** Object trajectories, discovery moments
- **Accessibility:** WCAG AAA compliant with black text

### Functional Colors

- **Success Green:** `#10B981` - Confirmed detections
- **Warning Amber:** `#F59E0B` - Pending confirmations
- **Error Red:** `#EF4444` - Critical alerts
- **Info Blue:** `#3B82F6` - Educational content

### Dark Mode (Default)
- Background: Cosmic Deep Blue (#0A1628)
- Text: Starlight White (#F5F7FA)
- Optimized for nighttime observation and stargazing

### Light Mode (Alternative)
- Background: Starlight White (#F5F7FA)
- Text: Cosmic Deep Blue (#0A1628)
- Available for daytime use

## Typography System

### Font Families

#### Primary: Inter
- **Weights:** 400 (Regular), 500 (Medium), 600 (Semibold), 700 (Bold)
- **Usage:** All UI text, body copy, navigation
- **Rationale:** Exceptional mobile legibility, professional appearance

#### Display: Space Grotesk
- **Weights:** 500 (Medium), 700 (Bold)
- **Usage:** Headlines, hero text, data callouts
- **Rationale:** Distinctive personality, geometric precision

#### Monospace: JetBrains Mono
- **Weight:** 400 (Regular), 500 (Medium)
- **Usage:** Coordinates, timestamps, technical data
- **Rationale:** Clear technical information display

### Type Scale (Mobile-Optimized)

**Hero (H1)**
- Font: Space Grotesk Bold
- Size: 32px / 2rem
- Line Height: 1.2
- Use: Landing hero, major headers

**Large Heading (H2)**
- Font: Space Grotesk Bold
- Size: 24px / 1.5rem
- Line Height: 1.3
- Use: Page titles, major sections

**Medium Heading (H3)**
- Font: Inter Semibold
- Size: 20px / 1.25rem
- Line Height: 1.4
- Use: Card titles, subsections

**Body Large**
- Font: Inter Regular
- Size: 16px / 1rem
- Line Height: 1.6
- Use: Primary content, articles

**Body Regular**
- Font: Inter Regular
- Size: 14px / 0.875rem
- Line Height: 1.6
- Use: Card content, secondary text

**Body Small**
- Font: Inter Regular
- Size: 12px / 0.75rem
- Line Height: 1.5
- Use: Labels, captions, metadata

## Logo Guidelines

### Logo Concept
The ISO Tracker logo represents the intersection of discovery, precision, and cosmic wonder through:
1. Interstellar Journey - Visual suggestion of objects traveling through space
2. Tracking Precision - Technical design suggesting measurement accuracy
3. Cosmic Scale - Elements evoking the vastness of space

### Primary Logo
**Wordmark + Icon Combination**
- Icon: Stylized orbital trajectory with tracking elements
- Wordmark: "ISO TRACKER" in Space Grotesk Bold
- Secondary text: "Interstellar Object Tracker" (optional)

### Color Variations
1. **Dark Mode:** Icon in Nebula Blue, text in Starlight White
2. **Light Mode:** Icon in Cosmic Deep Blue, text in Cosmic Deep Blue
3. **Monochrome Dark:** All elements in Starlight White
4. **Monochrome Light:** All elements in Cosmic Deep Blue

### Minimum Sizes
- **Full Logo:** 140px width minimum
- **Icon Only:** 32×32px minimum
- **Wordmark Only:** 120px width minimum

### Clear Space
Space around logo equal to height of the "I" in "ISO"

### Usage Rules

**Do:**
- Use approved color variations only
- Maintain aspect ratio when scaling
- Place on high-contrast backgrounds
- Use icon-only version for favicons

**Don't:**
- Rotate, skew, or distort the logo
- Add effects (shadows, glows, gradients)
- Place on busy backgrounds
- Rearrange elements
- Recreate or modify

## Visual Elements

### Photography Guidelines

**Primary Image Types:**
1. **Space Photography** - Real astronomical imagery from NASA/ESA
2. **Data Visualizations** - Clean, scientifically accurate charts
3. **UI Screenshots** - Actual platform functionality

**Image Treatment:**
- Cool color temperature for space aesthetic
- Subtle blue shift for brand consistency
- Maintain natural look, avoid heavy stylization

**Resolution Standards:**
- Mobile: Minimum 750px width
- Desktop: Minimum 1920px width
- Thumbnails: Minimum 300px
- File Size: Optimize to <300KB for mobile

### Iconography System

**Style:** Outline icons with 2px stroke, geometric precision

**Standards:**
- Size: 24×24px base grid
- Stroke: 2px uniform weight
- Corners: 2px radius
- Color: Inherits from parent or Nebula Blue for active

**Icon Categories:**
- Navigation (home, search, profile, settings)
- Object Types (asteroid, comet, unknown)
- Actions (share, save, filter, refresh)
- Data (velocity, distance, trajectory)

## Layout & Composition

### Mobile-First Design

All layouts begin with mobile optimization (375px primary target)

#### Screen Breakpoints
- **Mobile:** 375px - 767px (Primary)
- **Tablet:** 768px - 1023px (Secondary)
- **Desktop:** 1024px+ (Enhancement)

### Spacing System
**Base Unit:** 4px - All spacing uses 4px increments

**Scale:**
- xs: 4px - Tight spacing
- sm: 8px - Related elements
- md: 16px - Standard padding
- lg: 24px - Section spacing
- xl: 32px - Page spacing
- 2xl: 48px - Major sections
- 3xl: 64px - Hero sections

### Component Standards

**Cards:**
- 16px padding all sides
- 8px border radius
- Subtle shadow: 0 2px 8px rgba(0,0,0,0.1)
- Minimum 44px touch targets

**Lists:**
- 56px minimum height per item
- 16px horizontal padding
- 12px vertical padding

**Navigation:**
- 56px fixed height header
- Bottom tab bar for mobile
- 44×44px minimum touch targets

## Application Guidelines

### Website

**Homepage:**
- Hero: Full-viewport with live object position
- Features: Card-based grid (1 col mobile, 3 col desktop)
- Footer: Dark background, stacked on mobile

**Object Detail:**
- Back button + object name header
- Key metrics at top (distance, velocity)
- Trajectory visualization
- Technical details (expandable)
- Related objects carousel

### Mobile PWA

**Tab Navigation:**
1. **Home** - News feed and discovery updates
2. **Evidence** - Evidence framework dashboard and analysis
3. **Track** - Real-time ISO position tracking
4. **Community** - Discussion, debate, and peer validation
5. **Account** - Profile and settings

**Standards:**
- Bottom tab bar always visible
- Active tab: Nebula Blue
- Icons + labels for clarity
- 56px height

**Key Feature Screens:**
- **Evidence Framework Dashboard** - Community Sentiment vs. Scientific Consensus
- **Evidence Timeline** - Observation → Hypothesis → Theory hierarchy
- **Historical ISO Comparison** - 1I/'Oumuamua, 2I/Borisov, 3I/ATLAS analysis
- **Educational Modules** - ISO detection science, evaluation methodology

### Email

**Template Structure:**
- Header: Logo, centered, max 200px
- Body: Max 600px width, 24px padding
- CTA: Large button, Nebula Blue, centered
- Footer: Unsubscribe, social, address

**Requirements:**
- Minimum 16px body text
- Alt text for all images
- High contrast (WCAG AA minimum)
- Mobile-responsive
- Plain text alternative

### Social Media

**Platform Standards:**
- Twitter/X: 1200×628px cards
- Instagram: 1080×1080px posts, 1080×1920px stories
- Facebook: 1200×630px posts
- LinkedIn: Professional tone, data-focused
- YouTube: 1280×720px thumbnails

**Assets:**
- Profile: Icon-only logo
- Posts: Cosmic aesthetic, brand colors
- Consistent cool tone filter

## Brand Voice

### Personality Traits
- **Scientific** - Rigorous, evidence-based
- **Accessible** - Clear, jargon-free
- **Wonder-Inspiring** - Maintains cosmic awe
- **Trustworthy** - Honest about uncertainties
- **Collaborative** - Community-driven

### Voice Characteristics
- **Evidence over speculation** - Always ground claims in data
- **Rigor over hype** - Scientific credibility, not sensationalism
- Curious but not sensational
- Precise but not academic
- Enthusiastic but not hyperbolic
- Inclusive but not patronizing

### Core Values
1. **Intellectual Honesty** - Follow data, admit unknowns
2. **Curiosity Over Certainty** - Question everything
3. **Simplicity & Clarity** - Make complex ideas accessible
4. **Rigorous Method** - Systematic observation over speculation
5. **Collaborative Discovery** - Build on collective intelligence

## Accessibility Standards

### WCAG AAA Compliance

**Visual:**
- Text contrast: 7:1 minimum (AAA standard)
- UI components: 3:1 minimum
- Touch targets: 44×44px minimum
- Color not sole information carrier

**Content:**
- Alt text for all images
- Proper heading hierarchy
- Descriptive link text
- Keyboard navigation support

**Technical:**
- Responsive 320px to 2560px
- Text resizable to 200%
- Valid semantic HTML
- ARIA labels where needed
